## 定义

数据集（Dataset）是用于训练、验证和评估机器学习模型的结构化数据集合。在AI和机器学习语境下，数据集由多个数据点（也称样本、实例或样本）组成，每个数据点包含一组特征（输入变量），在监督学习场景下还包含对应的标签（输出变量）。数据集是机器学习的基础——模型只能学到数据中存在的模式，因此数据集的质量、规模和代表性直接决定了模型的性能与可靠性。

|------|------|

- **Set**: /sɛt/

### 逐词解释

| 单词 | 词源 | 中文解释 |
|------|------|----------|
| Data | 拉丁语 *datum*（给予之物），*data* 为其复数形式 | 数据；被收集、记录用于分析的信息，可以是数字、文本、图像等任何形式 |
| Set | 古英语 *settan*（放置、安置） | 集合；具有某种共同特性的元素的汇总，数学中指一组确定的、互不相同的对象 |

### 组合理解

"Dataset" 字面含义为"数据的集合"。在机器学习领域，它特指为了特定分析或建模目的而被组织在一起的数据集合体。该词强调的不仅是数据的简单堆叠，更是数据经过组织、结构化后形成的可分析整体。正如哲学学者Sabina Leonelli所指出，数据是一种"关系性类别"——什么算作数据取决于谁在使用、如何使用以及为了什么目的。[(arXiv:2501.15491)](https://arxiv.org/pdf/2501.15491) 如今"dataset"（单词形式）已成为技术文献中的标准写法，尽管"data set"（两词形式）在某些正式和统计学语境中仍可见到。

## 详细说明

### 数据集的基本构成

| 构成要素 | 英文 | 说明 |
|----------|------|------|
| 样本 | Sample / Instance | 数据集中的每个独立数据项，如一张图片、一段文本 |
| 特征 | Feature | 描述样本的属性或变量，是模型的输入 |
| 标签 | Label | 监督学习中每个样本对应的正确输出 |
| 元数据 | Metadata | 描述数据来源、采集时间、格式等的信息 |

[(IBM - What is a dataset)](https://www.ibm.com/think/topics/dataset) [(arXiv:2501.15491)](https://arxiv.org/pdf/2501.15491)

### 按用途划分

在机器学习项目中，数据集通常被划分为三个互不重叠的子集：

| 类型 | 英文 | 用途 | 典型占比 |
|------|------|------|----------|
| 训练集 | Training Set | 用于拟合模型参数，模型从中学习模式 | 60%-80% |
| 验证集 | Validation Set | 用于超参数调优和模型选择，检测过拟合 | 10%-20% |
| 测试集 | Test Set | 用于最终评估模型泛化能力，完全不参与训练 | 10%-20% |

用学习类比：训练集是学生的课本，验证集是模拟考试，测试集是期末考试。测试集必须严格与训练过程隔离，否则评估结果将失去意义。

### 按数据结构划分

| 类型 | 说明 | 示例 |
|------|------|------|
| 结构化数据集 | 以表格形式组织，有明确的行列定义 | CSV文件、SQL数据库 |
| 半结构化数据集 | 有一定格式标记但不够严格 | JSON、XML文件 |
| 非结构化数据集 | 无预定义格式 | 图像、音频、视频、原始文本 |

### 按标注状态划分

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 标注数据集 | 每个样本都有对应的正确标签 | 监督学习 |
| 未标注数据集 | 样本没有标签 | 无监督学习 |
| 半标注数据集 | 部分样本有标签 | 半监督学习 |

### 著名数据集

- **MNIST**：手写数字图像数据集，机器学习入门经典
- **ImageNet**：超过1400万张标注图像，推动了深度学习在计算机视觉领域的突破
- **Common Crawl**：PB级网页爬取数据，许多大语言模型的训练数据来源
- **CIFAR-10**：10类6万张小图像，图像分类基准

### 核心原则

**"垃圾进，垃圾出"（Garbage In, Garbage Out）**：即使是最先进的算法，也无法从低质量的数据中学到有意义的模式。数据集的质量是决定AI系统成败的第一要素。

## 相关概念

- **数据预处理（Data Preprocessing）**：在将数据输入模型之前进行的清洗、转换和标准化操作，包括处理缺失值、去除异常值、特征缩放等。数据预处理的质量直接影响模型的训练效果。

- **数据增强（Data Augmentation）**：通过对现有样本进行变换（如旋转、裁剪、同义改写等）来生成新训练样本的技术，用于扩大数据集规模和多样性，缓解数据不足的问题。

- **基准数据集（Benchmark Dataset）**：为公平评估和比较不同模型而设计的标准化数据集，如ImageNet、MMLU等。它们为研究社区提供共同的评估基础，使不同方法的结果具有可比性。

- **合成数据（Synthetic Data）**：通过算法（如GAN、VAE、LLM）人工生成的数据，用于弥补真实数据的不足、保护隐私或平衡数据集。到2024年，据估计超过60%的AI训练数据已是合成数据。

- **数据表（Datasheet）**：由Timnit Gebru等人提出的数据集文档规范，要求记录数据集的特征、组成、收集动机、推荐用途和伦理考虑，类似于电子产品的规格说明书。[(arXiv:2501.15491)](https://arxiv.org/pdf/2501.15491)
