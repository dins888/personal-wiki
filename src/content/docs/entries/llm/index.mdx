# 🤖 大语言模型

大语言模型（LLM）相关的核心概念。

- [大语言模型（LLM）](./llm/) — 基于海量文本训练的深度学习语言模型
- [词元（Token）](./token/) — 大语言模型处理文本的最小语义单元
- [上下文窗口](./context-window/) — 模型一次能处理的最大文本长度
- [提示词（Prompt）](./prompt/) — 用户输入给模型的文本指令
- [稀疏（Sparse）](./sparse/) — 大部分元素为零或仅部分参与计算
- [稠密（Dense）](./dense/) — 所有元素非零或全部参与计算
