# 📚 AI 基础概念

人工智能领域的核心基础概念。

- [人工智能（AI）](./ai/) — 让计算机模拟人类智能行为的技术总称
- [机器学习](./machine-learning/) — 让计算机从数据中自动学习规律的技术
- [深度学习](./deep-learning/) — 基于多层神经网络的机器学习分支
- [神经网络](./neural-network/) — 模仿生物神经网络结构的计算模型
- [生成式 AI](./generative-ai/) — 能够生成新内容的 AI 系统
- [基础模型](./foundation-model/) — 在大规模数据上预训练的通用 AI 模型
- [多模态 AI](./multimodal-ai/) — 能同时处理文本、图像、语音等信息的 AI
