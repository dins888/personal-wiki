# 🔧 技术应用

AI 技术及常用开发者工具在实际应用中的核心概念。

## AI 与智能体

- [提示工程](./prompt-engineering/) — 系统性地设计和优化提示词的技术
- [思维链](./chain-of-thought/) — 引导模型展示推理过程的提示技术
- [AI Agent](./agent/) — 能自主规划、调用工具、完成复杂任务的 AI 系统
- [检索增强生成（RAG）](./rag/) — 将外部知识检索与模型生成相结合的技术

## AI 工具与客户端

- [扣子（Coze）](./coze/) — 字节跳动的一站式 AI Bot 开发平台
- [Cherry Studio](./cherry-studio/) — 开源多模型 AI 聊天客户端
- [Hermes](./hermes/) — 隐私优先的轻量级 AI 聊天客户端

## 开发工具

- [Visual Studio Code](./visual-studio-code/) — 微软开源的轻量级代码编辑器
- [Postman](./postman/) — API 开发和测试工具
- [Git](./git/) — 分布式版本控制系统
- [Python 3.1](./python31/) — Python 编程语言的 3.1 版本

## 数据库与容器

- [Docker](./docker/) — 开源容器化平台
- [Docker Desktop](./docker-desktop/) — Docker 官方桌面应用
- [MongoDB Compass](./mongodb-compass/) — MongoDB 官方图形化管理工具
- [Neo4j Desktop](./neo4j-desktop/) — Neo4j 图数据库本地图形管理应用
- [Attu](./attu/) — Apache Kafka 集群管理监控 GUI

## 系统工具与效率

- [CLI（命令列接口）](./cli/) — 通过纯文本命令与计算机交互的界面
- [API](./api/) — 应用程序编程接口，软件系统间的通信契约
- [CC Switch](./cc-switch/) — macOS 中英文输入法快速切换工具
- [uTools](./utools/) — 跨平台效率工具启动器
- [WizTree](./wiztree/) — 磁盘空间极速分析工具
- [Legion Zone](./legion-zone/) — 联想 Legion 游戏本控制中心
- [康威定律（Conway's Law）](./conways-law/) — 系统架构必然反映组织沟通结构的经验法则
- [V2EX](./v2ex/) — 面向程序员和创意工作者的中文技术社区
