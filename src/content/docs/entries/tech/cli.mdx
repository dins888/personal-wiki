## 定义

**CLI**（Command Line Interface，命令列接口）是一种通过纯文本命令与计算机程序进行交互的用户界面。用户在终端（Terminal）或命令行（Command Prompt）中输入特定命令，程序解析并执行后返回文本结果。

与之相对的是 **GUI**（Graphical User Interface，图形用户界面），后者通过鼠标点击、按钮等可视化元素进行交互。

## 英文术语拆解

| 英文全称 | Command Line Interface |
|----------|----------------------|
| 音标 | /kəˈmænd laɪn ˈɪntərfeɪs/ |

**逐词解释：**

| 单词 | 音标 | 含义 |
|------|------|------|
| Command | /kəˈmænd/ | 命令、指令。指用户输入给计算机的一条操作指令。 |
| Line | /laɪn/ | 行、线路。在这里指"一行文本"——用户在命令行中输入的一行文字。 |
| Interface | /ˈɪntərfeɪs/ | 接口、界面。指人与机器之间进行交互的媒介或通道。 |

**组合理解：** "Command Line"（命令行）+ "Interface"（界面）= 通过文本行输入命令的交互界面。

## 常见用法示例

```bash
# 查看当前目录下的文件
ls -la

# 创建新目录
mkdir my-project

# 进入目录
cd my-project
```

## 相关概念

- **Shell**：命令解释器，负责解析和执行 CLI 命令（如 Bash、Zsh、PowerShell）
- **Terminal**：终端，运行 Shell 的窗口程序
- **GUI**（Graphical User Interface）：图形用户界面，CLI 的对立面
