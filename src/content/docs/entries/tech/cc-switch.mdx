## 定义

**CC Switch** 是一个 macOS 平台上的**系统工具**（System Utility），用于在**中文输入法**（如系统自带的拼音输入法或第三方输入法）和**英文输入法**之间进行快速切换。它通过快捷键绑定或菜单栏图标（Menu Bar Icon）实现输入法切换的便捷操作，解决 macOS 用户在日常使用中频繁切换中英文输入法的痛点。

## 英文术语拆解

| 英文全称 | CC Switch |
|----------|-----------|
| 音标 | /siː siː swɪtʃ/ |

**逐词解释：**

- **CC**：通常指 "Chinese Character" 或 "Chinese/English" 的缩写，表示中文与英文之间的切换场景
- **Switch**：/swɪtʃ/，意为"开关、切换"，指在两种输入法状态之间进行切换操作

**组合理解：** CC Switch 即"中英文切换工具"，是一个专注于简化 macOS 上中英文输入法切换体验的小型实用工具。

## 核心功能/特性

### 快捷键切换
通过自定义快捷键（如 Control + Space 或其他组合键）一键切换中英文输入法，无需打开系统偏好设置。

### 菜单栏图标
在 macOS 菜单栏显示一个状态图标，直观展示当前输入法状态（中文/英文），点击即可切换。

### 状态指示
清晰显示当前激活的输入法类型，帮助用户确认当前输入状态，避免输入错误。

### 轻量无感
作为 macOS 原生工具或轻量级应用运行，不占用过多系统资源，启动后在后台静默运行。

## 常见用法示例

### 典型使用场景

1. **写作切换**：在编写文档时，需要频繁在中文正文和英文代码/术语之间切换输入法，通过 CC Switch 的快捷键一键完成
2. **编程场景**：在 IDE 中输入英文代码，切换到注释区需要中文输入，用快捷键快速切换
3. **聊天场景**：在即时通讯工具中，中英文混合输入时快速切换

### 配置示例

```
# 典型快捷键配置
中文输入法 ↔ 英文输入法：Control + Space
显示菜单栏图标：开启
启动时默认输入法：英文
```

## 相关概念

- **输入法（Input Method Editor, IME）**：将键盘输入转换为特定语言文字的中间层程序
- **macOS 输入法框架**：Apple 提供的系统级输入法管理框架
- **Caps Lock 切换**：macOS 系统支持的通过 Caps Lock 键切换中英文的功能
- **Karabiner-Elements**：macOS 上的键盘自定义工具，可增强输入法切换体验
- **搜狗输入法/百度输入法**：第三方中文输入法，部分自带切换增强功能
