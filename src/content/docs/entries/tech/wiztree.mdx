## 定义

WizTree 是一款 Windows 平台上的磁盘空间分析工具，由 Ian Bousfield 开发，由 Antibody Software 发布。它能够以极快的速度扫描并可视化展示硬盘上各文件和文件夹的大小分布。与其他同类工具（如 WinDirStat、TreeSize）不同，WizTree 直接读取 NTFS 文件系统的主文件表（MFT，Master File Table）来获取文件信息，而非逐文件遍历整个磁盘树，因此扫描速度通常比同类工具快数十倍——扫描一个数百 GB 的分区往往只需几秒钟。WizTree 提供树状图（Treemap）和文件列表两种可视化视图，帮助用户快速定位占用大量空间的"大文件"，从而高效清理磁盘空间。

## 核心功能/特性

### 1. 极速 MFT 扫描

WizTree 的核心技术优势在于直接读取 NTFS 文件系统的 MFT（Master File Table，主文件表）。MFT 是 NTFS 分区中存储所有文件和目录元数据（文件名、大小、时间戳等）的数据库。传统工具需要递归遍历整个目录树来收集文件信息，而 WizTree 直接解析 MFT 记录，跳过文件系统遍历开销，因此扫描速度远超同类工具。

- **扫描速度：** 数百 GB 的分区通常只需几秒即可完成扫描
- **对比：** 比 WinDirStat 快 10-100 倍（取决于文件数量和磁盘类型）
- **限制：** MFT 快速扫描仅适用于 NTFS 分区；对于 FAT32/exFAT 等非 NTFS 分区，WizTree 回退为传统的目录遍历模式，速度会显著降低

### 2. 树状图可视化（Treemap）

WizTree 使用 Treemap（树状图）来直观展示磁盘空间占用情况：

- **矩形面积**代表文件/文件夹占用的空间大小
- **颜色编码**区分不同的文件类型或文件夹层级
- **支持钻取：** 点击某个矩形区域可以深入查看其子目录结构
- **大文件一目了然：** 占用空间最大的文件在图中显示为最大的矩形块

### 3. 文件列表视图

除 Treemap 外，WizTree 还提供传统的分层文件列表视图：

- 以树形结构展示文件夹及其子文件夹的大小
- 显示每个文件夹包含的文件数量
- 支持按大小、文件数量、修改日期等排序
- 展示文件扩展名分布统计

### 4. 文件过滤与搜索

- 支持按文件扩展名过滤（如仅显示 .mp4 文件）
- 支持按文件大小范围过滤
- 支持按修改日期过滤
- 快速定位特定类型的占空间大文件

### 5. 其他实用功能

- **导出功能：** 可将扫描结果导出为 CSV 格式，方便进一步分析
- **便携版：** 提供绿色免安装版本（Portable），可放在 U 盘中随身携带
- **多语言支持：** 支持包括简体中文在内的多种语言界面
- **免费供个人使用：** 个人用户免费，商业使用需购买许可

## 常见用法示例

### 示例 1：快速找出占空间最大的文件

1. 启动 WizTree，选择需要分析的磁盘分区（如 C:）
2. 点击"Scan"按钮，等待数秒完成 MFT 扫描
3. 在 Treemap 视图中，面积最大的矩形色块即为占空间最大的文件
4. 右键点击该文件，可选择"Open"在资源管理器中定位，或查看其详细信息

### 示例 2：分析某个文件夹的空间占用

1. 扫描完成后，在左侧的树形目录列表中展开目标文件夹
2. 查看右侧 Treemap 自动聚焦于所选文件夹的子结构
3. 切换到"File List"标签页，按大小降序排列，找出该文件夹中最大的文件

### 示例 3：按文件类型筛选

1. 在扫描界面的过滤栏中输入扩展名（如 `*.log`）
2. 视图自动刷新，仅显示匹配的文件
3. 快速定位日志文件、临时文件等可清理文件占用的总空间

### 示例 4：清理磁盘前的对比分析

1. 清理前运行 WizTree，导出扫描结果为 CSV
2. 删除不需要的文件后再次运行扫描
3. 对比两次结果，确认清理效果

## 相关概念

- **主文件表（MFT，Master File Table）：** NTFS 文件系统的核心数据结构，记录分区上所有文件和目录的元数据。WizTree 通过直接读取 MFT 实现极速扫描，这是其区别于同类工具的关键技术。
- **NTFS（New Technology File System）：** Windows NT 系列操作系统的默认文件系统，支持大文件、文件权限、加密、压缩等高级特性。MFT 是 NTFS 的标志性特征。
- **WinDirStat：** 另一款经典的 Windows 磁盘空间分析工具，采用开源方式发布，功能与 WizTree 类似但扫描速度远慢于 WizTree（因其采用传统的目录遍历方式）。
- **TreeSize：** 另一款流行的 Windows 磁盘分析工具，提供免费版和专业版，有命令行版本可用于自动化场景。
- **Treemap（树状图）：** 一种数据可视化方法，由 Ben Shneiderman 于 1990 年发明，使用嵌套矩形的面积来表示层级结构中的数值大小，非常适合展示文件空间占用。
- **磁盘清理（Disk Cleanup）：** 操作系统内置的磁盘空间管理功能，WizTree 可帮助用户在清理前精确定位需要清理的文件。
