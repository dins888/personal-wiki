## 定义

**uTools** 是一个国产的**跨平台效率工具启动器**（支持 Windows、macOS、Linux）。通过全局快捷键（默认 Alt+Space）唤出搜索框，支持快速启动应用、文件搜索、代码片段管理、JSON 格式化、二维码生成、图片压缩、正则表达式测试等数十种插件功能。采用插件化架构，用户可按需安装功能模块。

uTools 的定位类似于 macOS 上的 Spotlight 或 Alfred，但更加聚焦于开发者和技术用户的效率提升场景，通过丰富的插件生态满足多样化需求。


## 第一性

|----------|--------|
| 音标 | /juː tuːlz/ |

**逐词解释：**

- **u**：代表 "your"（你的），也有 "you"（你）的含义，体现以用户为中心的理念。
- **Tools**：/tuːlz/，"工具"的复数形式，表示这是一套工具集合。

**组合理解：** "uTools" 即"你的工具集"，寓意这是一个可以按个人需求自由组合的效率工具集合，强调个性化和可定制性。

## 核心功能

### 全局快捷搜索
通过 Alt+Space（可自定义）唤起搜索面板，支持快速搜索应用、文件、剪贴板历史等内容，无需切换窗口。

### 插件化架构
核心是一个轻量级的启动框架，所有功能通过插件扩展。用户可在插件市场中按需安装，不需要的功能不会占用资源。

### 常用开发者工具
内置或可通过插件获得多种开发者常用工具，包括 JSON 格式化、正则表达式测试、时间戳转换、编解码（Base64、URL）、颜色转换、UUID 生成等。

### 文件搜索
基于索引的文件快速搜索，支持按文件名、文件类型、内容等条件搜索本地文件。

### 代码片段管理
支持保存和管理常用的代码片段，通过关键词快速检索和插入。

### 剪贴板历史
记录剪贴板使用历史，支持快速回溯和重新复制。

### 窗口管理与快速启动
支持快速启动已安装的应用程序，部分插件提供窗口管理（如窗口切换、分屏）功能。

### 超级面板
长按快捷键可呼出超级面板，提供基于当前上下文的快捷功能（如在文件管理器中显示文件操作选项，在浏览器中显示翻译选项）。

## 常见用法示例

```text
# 典型使用场景

1. 快速计算：唤起 uTools → 输入 "128 * 256" → 即时显示结果
2. JSON 格式化：复制 JSON → 唤起 uTools → 选择 "JSON 格式化" 插件
3. 正则测试：唤起 uTools → 输入正则和测试文本 → 即时匹配结果
4. 颜色转换：输入 "#FF5733" → 自动显示 RGB、HSL 等格式
5. 快速启动：输入应用名前几个字母 → 回车启动应用
6. 编码转换：选中文字 → 唤起超级面板 → 选择 Base64 编码/解码
```

## 相关概念

- **Alfred**：macOS 上知名的效率启动器，uTools 的灵感来源之一
- **Everything**：Windows 上极速文件搜索工具
- **Spotlight**：macOS 内置的系统搜索功能
- **Wox**：Windows 上的开源启动器（已停止维护）
- **Raycast**：macOS 上的新一代效率工具，类似 Alfred 的现代替代品
- **插件化架构（Plugin Architecture）**：通过插件扩展核心功能的软件设计模式
