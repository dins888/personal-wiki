## 定义

**Legion Zone** 是联想（Lenovo）为其 **Legion 游戏本产品线**提供的**系统控制软件**（System Control Software）。用户可通过该软件调节性能模式（安静/均衡/性能）、风扇转速、RGB 灯效、屏幕刷新率等硬件参数，是联想 Legion 游戏本的**专属控制中心**（Control Center）。


## 第一性

> 让一台笔记本在安静与狂暴之间一键切换的性能调度器。

## 比方

> 像汽车上的驾驶模式旋钮—— Economy、Normal、Sport 转一下，同一台车就像换了一副脾气。

## 英文术语拆解

| 英文全称 | Legion Zone |
|----------|-------------|
| 音标 | /ˈliːdʒən zoʊn/ |

**逐词解释：**

- **Legion**：/ˈliːdʒən/，意为"军团、大批"，是联想游戏本产品线的品牌名称。该名称传达力量感和规模感，暗示"军团般"的强劲性能
- **Zone**：/zoʊn/，意为"区域、地带"，暗示这是一个专属的控制空间/领域

**组合理解：** Legion Zone 即"军团地带/军团空间"，是联想 Legion 游戏本用户的专属控制区域，在这个软件空间中管理设备的一切硬件设置。

## 核心功能/特性

### 性能模式切换
提供三种或更多预设性能模式：
- **安静模式（Quiet Mode）**：降低 CPU/GPU 功耗和风扇转速，适合办公和图书馆场景
- **均衡模式（Balance Mode）**：自动调节性能和噪音的平衡，适合日常使用
- **性能模式（Performance Mode）**：最大化 CPU/GPU 功耗释放，适合游戏和重度计算
- **自定义模式（Custom Mode）**：允许用户手动调节功耗和风扇曲线

### 风扇转速控制
支持手动调节风扇转速，查看 CPU 和 GPU 的温度状态，设置自定义风扇曲线。

### RGB 灯效管理
控制键盘背光和机身 RGB 灯带的颜色、亮度和灯效模式（呼吸、渐变、常亮等）。

### 屏幕刷新率调节
支持在不同刷新率（如 60Hz、144Hz、165Hz、240Hz）之间切换，平衡画面流畅度和电池续航。

### 系统监控
实时显示 CPU 温度、GPU 温度、风扇转速、CPU/GPU 功耗等硬件状态信息。

### 超频/降压
高级用户可对 CPU 和 GPU 进行超频或降压调节，优化性能或能效表现。

## 常见用法示例

### 典型使用场景

1. **游戏前**：切换到性能模式，确保 CPU 和 GPU 满血运行，获得最高帧率
2. **办公/会议**：切换到安静模式，降低风扇噪音，延长电池续航
3. **日常使用**：均衡模式自动调节，兼顾性能和静音
4. **个性化**：自定义键盘 RGB 灯效，设置喜欢的颜色和灯效模式
5. **性能调优**：手动调整风扇曲线，在游戏时让风扇更早提速散热

### 性能模式对比

| 模式 | CPU 功耗 | GPU 功耗 | 风扇噪音 | 适用场景 |
|------|---------|---------|---------|---------|
| 安静 | 低 | 低 | 极低 | 办公、图书馆、会议 |
| 均衡 | 中 | 中 | 中 | 日常使用、轻度游戏 |
| 性能 | 高 | 高 | 高 | 3A 游戏、视频渲染、AI 训练 |

## 相关概念

- **联想 Legion 系列**：联想游戏本产品线，包括 Legion 5、Legion 7、Legion Pro 等型号
- **性能释放（TDP/TGP）**：CPU 和 GPU 的功耗释放水平，直接影响性能表现
- **AWCC（Alienware Command Center）**：戴尔外星人游戏本的类似控制软件
- **Armoury Crate**：华硕游戏本的控制中心软件
- **MSI Center**：微星游戏本的控制软件
- **HWMonitor / HWiNFO**：第三方硬件监控工具，可补充 Legion Zone 的监控功能
