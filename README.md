# 个人百科知识库

> 持续积累，科学查证，版本化管理的个人知识百科。

## 技术栈

- **Astro + Starlight** — 静态站点生成框架
- **GitHub Pages** — 免费托管和自动部署
- **GitHub Actions** — push 即部署

## 目录结构

```
src/content/docs/
├── entries/          ← 百科词条（核心内容）
├── categories/       ← 分类索引页
├── templates/        ← 词条模板
└── about.mdx         ← 项目说明
```

## 查证工作流

每个词条经过信源分级检索 + 三角验证：

1. **摄入** → 用户提出概念
2. **检索** → T1-T5 信源等级分层检索
3. **审核** → 三角验证（≥3 个独立来源）
4. **确认** → 用户最终判断
5. **沉淀** → 写入百科

快速通道：常见概念 1 个 T1/T2 来源即可。
